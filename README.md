# Mower2 NodeJS


### To run application:

Prerequisites

  * Install modules with npm

With scrip shell:

```sh
./run.sh -o
```

With grunt:

```sh
grunt
```

### To run server:

Prerequisites

  * Install modules with npm

With scrip shell:

```sh
./run.sh
```

### To run the tests:

Prerequisites

  * Run server

With grunt:

```sh
grunt test
```
